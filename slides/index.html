Understanding the jQuery Object

$("div").addClass("special");

  1. $ - is the jQuery Object (also name jQuery)
  2. "div" - the name of an element using CSS selector
  3. $("div") - represents every div on the page
  4. addClass() - modifier method for the element
  5. "special" - can be anything that executes as a result of the element being modified

Graceful Death

$(".idontexist").addClass("special");

will not break the code.

Ready Catcher

In order to traverse and manipulate the page we must wait until it's ready to be used. jQuery has a ready event that fires the instant the DOM is ready to be worked with. Stick all your jQuery code in a ready block, load it unobtrusively.

The Ready Event


$(document).ready(function(){ 
 // Your jQuery code goes in here 
});

Alternative - The Ready Event

Use of Anonymous self calling functions


(function($){ 
 // Your jQuery code goes in here
})(jQuery);
